ChofizzD 隱私權政策

最後更新日期：2026年7月8日

ChofizzD 非常重視您的隱私。本隱私權政策說明了本應用程式如何處理您的資料。

1. 資料收集

本應用程式致力於保護使用者的隱私。我們不會收集任何個人的身份識別資訊。您的運動紀錄、分數與相關資料均儲存於您的裝置本地端。

2. iCloud 同步

本 App 使用 Apple iCloud (CloudKit) 私有資料庫同步您的射擊賽紀錄、練習紀錄、地點與標籤等個人資料，僅用於在您登入相同 Apple ID 的多台裝置間同步顯示，資料儲存於您個人的 iCloud 帳戶中，開發者本身不會存取、蒐集或用於任何其他用途。您可隨時於裝置的「設定 > Apple ID > iCloud」中管理或關閉此 App 的 iCloud 同步權限。

3. 第三方服務

本應用程式不會將您的資料與任何第三方服務分享或用於廣告用途。

4. 隱私權保護措施

由於資料儲存於您的本地裝置或您私人的 iCloud 帳號中，我們建議您妥善保管您的裝置密碼與 Apple ID 帳號安全。

5. 聯絡我們
